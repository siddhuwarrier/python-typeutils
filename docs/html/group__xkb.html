Detailed Description

Package for wrapping C X Keyboard Function Calls.

This package was refactored from OSD Neo2 written by Martin Zuther (http://www.mzuther.de/en/contents/osd-neo2) and uses wrapped C XKB function calls to check the status of the Caps Lock and Num Lock keys. If anybody can think of a better name to give this package, please feel free to rename it!! :)
